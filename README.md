## 프로젝트 상세 설명

싸피생들만의 커뮤니티로 스터디 및 프로젝트를 모집하고 화상 회의 서비스를 이용하여 미팅을 진행할 수 있다. 시험/취업 정보를 공유할 수 있다. 자유 게시판을 통해 익명으로 서로 질문하고 답하며 필요 시 화상 회의 서비스 역시 이용할 수 있다.
